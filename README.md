# SafetyApp — Помощник по Охране труда

Веб-приложение для работы с памятками, ресурсами и тестами по охране труда.

## Возможности

- **Памятки** — справочные материалы по охране труда
- **Ресурсы** — полезные ссылки и документы (планируется)
- **Конструктор квестов** — создание интерактивных тестов и сценариев (планируется)
- **Тесты / квесты** — проверка знаний по охране труда (планируется)

## Технологии

- ASP.NET Core Razor Pages (.NET 10)
- Bootstrap 5 + jQuery
- Статический экспорт для GitHub Pages

## Запуск

```bash
# Режим разработки (веб-сервер)
dotnet run --project src/SafetyApp

# Экспорт в статический HTML
dotnet run --project src/SafetyApp -- --export
# Результат в publish/static/
```

После `--export` папку `publish/static/` можно залить на GitHub Pages или любой статический хостинг.

## Структура

```
src/SafetyApp/         — исходный код ASP.NET Core приложения
publish/static/        — сгенерированная статика (HTML+CSS+JS)
```

## Разработка

Проект использует Razor Pages. Добавляйте новые страницы в `src/SafetyApp/Pages/`. При экспорте все страницы автоматически обнаружатся и будут сконвертированы в HTML.
